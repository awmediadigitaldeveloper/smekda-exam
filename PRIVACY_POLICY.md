# Privacy Policy

## SMEKDA MOBILE TEST

Tanggal berlaku: 3 April 2026

SMEKDA MOBILE TEST adalah aplikasi Android yang digunakan untuk mengakses layanan web `https://smekda-mobile-test.vercel.app/` melalui antarmuka aplikasi mobile.

## Informasi yang Kami Kumpulkan

Aplikasi ini tidak membuat akun sendiri, tidak menjual data pengguna, dan tidak mengumpulkan data pribadi secara langsung di luar data yang diproses oleh layanan web yang dibuka di dalam aplikasi.

Data yang dapat diproses saat aplikasi digunakan:

- data yang dimasukkan pengguna pada website yang dibuka di dalam aplikasi
- file `.csv` yang dipilih pengguna secara manual saat menggunakan fitur upload
- data teknis dasar yang dibutuhkan agar halaman web dapat dimuat, seperti koneksi jaringan ke server tujuan

## Cara Aplikasi Mengakses File

Aplikasi tidak meminta izin akses penyimpanan secara luas.
Pemilihan file dilakukan hanya ketika pengguna secara aktif memilih file `.csv` melalui sistem file picker Android.

## Cara Kami Menggunakan Informasi

Informasi digunakan hanya untuk:

- memuat layanan web SMEKDA MOBILE TEST
- memungkinkan pengguna memilih dan mengunggah file `.csv`
- menjalankan fungsi dasar aplikasi seperti membuka tautan, memuat ulang halaman, dan berbagi tautan aplikasi

## Berbagi Data

Kami tidak menjual data pribadi pengguna kepada pihak ketiga.
Data yang dimasukkan atau diunggah pengguna melalui website tunduk pada pengelolaan layanan web SMEKDA MOBILE TEST.

## Keamanan

Kami berusaha menggunakan koneksi yang aman dan hanya memuat layanan web melalui HTTPS.

## Tautan ke Layanan Lain

Aplikasi dapat membuka tautan eksternal melalui browser atau aplikasi lain di perangkat ketika pengguna memilih tindakan tersebut atau saat halaman web memerlukannya.

## Anak-anak

Jika aplikasi ini digunakan oleh pelajar atau institusi pendidikan, pengelolaan data tetap mengikuti kebijakan layanan web yang terhubung dan kebijakan institusi terkait.

## Perubahan Kebijakan Privasi

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Versi terbaru harus dipublikasikan pada URL yang dicantumkan di Google Play Console.

## Kontak

Untuk pertanyaan terkait aplikasi ini, cantumkan email kontak resmi pengelola saat mempublikasikan aplikasi di Google Play Console.
