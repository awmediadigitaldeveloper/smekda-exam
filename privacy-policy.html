Privacy Policy

SMEKDA MOBILE TEST

Effective date: April 3, 2026

SMEKDA MOBILE TEST is an Android application used to access https://smekda-mobile-test.vercel.app/ through a mobile app interface.

Information We Collect

This app does not create its own user accounts, does not sell personal data, and does not directly collect personal information outside the data processed by the web service loaded inside the app.

Data that may be processed when using the app includes:

  • data entered by users on the website loaded inside the app
  • .csv files manually selected by users for upload
  • basic technical network data required to load the web service

File Access

The app does not request broad storage access. File selection is only performed when the user manually chooses a .csv file using the Android system file picker.

How Information Is Used

  • to load the SMEKDA MOBILE TEST web service
  • to allow users to select and upload .csv files
  • to support app actions such as refresh, open in browser, copy link, and share link

Data Sharing

We do not sell users' personal data. Any data entered or uploaded through the website is subject to the policies of the SMEKDA MOBILE TEST web service.

Security

We aim to use secure connections and load the connected web service over HTTPS.

External Links

The app may open external links in the device browser or other apps when required.
